Simple Notes App - Ocean Professional

- Routing: hash-based (#/, #/new, #/notes/:id)
- Persistence: localStorage key "notes_v1"
- Keyboard shortcuts:
  - Ctrl/Cmd+N: new note
  - Ctrl/Cmd+S: save
- Features:
  - Create, edit (live), delete with confirm and 10s undo
  - Pin/unpin, search, sort by updated desc with pinned on top
- Accessibility: semantic roles, labels, focus ring, respects reduced motion
